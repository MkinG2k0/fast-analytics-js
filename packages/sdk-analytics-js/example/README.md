# Пример использования SDK

Этот пример демонстрирует базовое использование Fast Analytics SDK.

## Запуск примера

1. Соберите SDK:

```bash
cd packages/sdk-analytics-js
npm run build
```

2. Откройте `example.html` в браузере (через локальный сервер):

```bash
# Используя Python
python -m http.server 8000

# Или используя Node.js
npx serve .
```

3. Откройте http://localhost:8000/example.html в браузере

4. Нажмите на кнопки для тестирования различных сценариев

## Важно

Перед использованием замените `your-project-api-key` на реальный API ключ вашего проекта из Fast Analytics панели.
