# Fast Analytics SDK Test App

Тестовое приложение на Vite + React для проверки работы SDK `@fast-analytics/sdk`.

## Установка

```bash
pnpm install
```

## Запуск

```bash
pnpm dev
```

Приложение откроется на `http://localhost:3001`

## Использование

1. Введите Project Key вашего проекта
2. Укажите Endpoint для отправки событий (по умолчанию: `http://localhost:3000/api/events`)
3. Нажмите "Инициализировать SDK"
4. Используйте кнопки для тестирования различных типов событий:
   - Error (Exception) - отправка ошибки через объект Error
   - Error (String) - отправка ошибки через строку
   - Async Error - отправка асинхронной ошибки
   - Warning - отправка предупреждения
   - Info - отправка информационного сообщения
   - Debug - отправка отладочного сообщения

## Особенности

- SDK автоматически перехватывает необработанные ошибки и промисы
- Все события отправляются батчами для оптимизации производительности
- Session ID генерируется автоматически и сохраняется в localStorage
